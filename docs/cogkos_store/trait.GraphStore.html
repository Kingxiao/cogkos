Graph activation diffusion - spread activation along graph edges

Implements the algorithm from ARCHITECTURE.md:

  • Vector-matched node A (activation=1.0) -> A –CAUSES–> B (weight=0.8) => B.activation += 1.0 * 0.8 * decay -> B –SIMILAR_TO–> C (weight=0.6) => C.activation += … -> Collect all nodes with activation > threshold
